What's inside

Every tool you reach for,
in one place

Everything you need to diagnose, map, and troubleshoot a network — in one window.

◎
Ping
Fixed and continuous ping with real-time RTT graph, packet loss tracking, and live results as each packet responds.
⊛
Multi-Ping
Monitor up to 5 hosts simultaneously. Cards turn green when responding, red when down. Keeps running while you use other modules.
⤵
Traceroute
Hop-by-hop path tracing with streaming results and color-coded latency per hop.
⊞
Subnet Sweep
Ping-sweep a range or CIDR subnet to discover live hosts. Supports /16 through /30 with paginated results and stop-scan control.
⊟
Subnet Calculator
Full CIDR breakdown — network address, broadcast, host range, usable hosts, and binary view.
⊕
IP Info
Detect your public IP, look up geolocation for any IP, and query WhoIs and RDAP records for any domain.
◈
DNS Lookup
Resolve A, AAAA, CNAME, MX, TXT, NS, and PTR records using Google, Cloudflare, or your own internal DNS server.
⊘
Port Scanner
TCP port scanning with pre-filled common ports, group presets (Web, Remote, Mail, Database), and custom port entry.
≋
Latency Guide
Reference tiers and per-application latency thresholds — what's acceptable for VoIP, gaming, RDP, and more.
